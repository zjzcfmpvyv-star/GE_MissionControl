# GoldenEra Mission Control

Enterprise-grade operational cockpit for GoldenEra Empire.

## Day-1 Scope (initialized)
- Frontend shell scaffolding (Next.js App Router structure)
- Domain architecture docs
- Initial PostgreSQL schema blueprint
- API contract starter (REST + realtime event channels)
- Local Docker compose baseline

## Planned Stack
- Next.js + Tailwind + shadcn/ui + Recharts
- Node API (Fastify/Nest style modules)
- Postgres + Redis + NATS
- WebSocket realtime

## Quick Start (after dependencies are added)
1. `cd mission-control`
2. `pnpm install`
3. `pnpm dev`
